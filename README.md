# Portfolio Personnel – Walae MORABET

## Présentation

Ce projet est un portfolio personnel présentant mon parcours, mes compétences transversales et mon projet professionnel.

## Contenu du projet

### index.html
Page d’accueil avec une présentation générale et une photo de profil.

### competences.html
Présentation de mes soft skills avec des exemples concrets :
- Gestion du temps
- Créativité
- Réflexivité
- Autonomie
- Esprit d’équipe
- Adaptation

### about.html
Présentation de mon parcours académique, de mes expériences, de mes centres d’intérêt et de mon projet professionnel.

### contact.html
Page permettant de me contacter grâce à un formulaire et à mes coordonnées.

### css/style.css
Contient la mise en forme et le design du site.

### js/script.js
Permet l’affichage dynamique des différentes compétences.

### assets/images
Contient les images utilisées dans le site.

## Technologies utilisées

- HTML
- CSS
- JavaScript
